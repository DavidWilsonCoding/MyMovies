myMovies

Description: The myMovies API returns information about movies including genre, length, release date and stars.

It also provides functionality for adding users and editing user information.

Business Logic URL Endpoint HTTP Method Query Parameters Request body data format Response body data format
Return a list of ALL movies to the user /movies GET NONE NONE A JSON object holding data about all the movies.
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:title GET title NONE A JSON object hodling data about the movie.
Example: 
            {
              title: '',
              genre: {
                ganreName: '',
                genreDesc: ''
              },
              director: {
                dirName: ''
              }
            }
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genres/:genre GET genre NONE A JSON object holding data about a genre.
Example: 
            {
              genreName: '',
              genreDesc: ''
            }
Return data about a director (bio, birth year, death year) by name /movies/directors/:dirName GET dirName NONE A JSON object holding data about a director.
Example: 
            {
              dirName: ''
            }
Allow new users to register /users POST genre A JSON object with user info.
Example: 
            {
              "userName": '',
              "topFilms": []
            }
A JSON object holding data about new user that was added.
Example: 
            {
              "userId": '',
              "userName": '',
              "topFilms": []
            }
Allow users to update their user info (userName) /users/:userId PUT userId A JSON object holding data about user's new info.
Example: 
            {
              userId: '',
              "userName": '',
              "topFilms": []
            }
A JSON object holding data about a user's changed info.
Example: 
            {
              "userName": '',
              "topFilms": []
            }
Allow users to add a movie to their list of favorites /users/:userId/:movie POST userId, movie A JSON object with movie info.
Example: 
            {
              title: '',
              genre: {
                genreNme: '',
                genreDesc: ''
              },
              director: {
                dirName: ''
              }
            }
A message stating the movie was added successfully (or not, if unsuccessful)
Allow users to remove a movie from their list of favorites /users/:userId/:movie DELETE userId, movie NONE A message stating the user's movie has been deleted (or not, if unsuccessful)
Allow existing users to deregister /users/:userId DELETE NONE NONE A message stating the user has been deleted (or not, if unsuccessful)
